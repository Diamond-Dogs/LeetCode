# 292. Nim Game
nim是很有意思的游戏，这个问题很经典，是零和博弈，题目简化了很多，因为两个人必定都会选取最优解，所以存在必胜和必败的情况，题目就是要求你找出先手必败的情况，然后就是找临界值，因为只能拿1-3颗石子，所以当对手拿完后石子剩下4颗时我们先手必败，分析清楚后实现就非常简单，只要n颗石子对4求余就好。
## 具体实现：
```
class Solution {
public:
    bool canWinNim(int n) {
        if(n%4==0)
            return false;
        return true;
    }
};
```
